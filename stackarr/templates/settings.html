Recommendations

Suggestions run on first login, then on this interval. Language filters which editions are suggested.

Auto-add to Chaptarr

Let Stackarr hand high-confidence picks to Chaptarr automatically, with no manual approval. Off by default. It skips books you already own, caps how many it adds per cycle, and pauses if Chaptarr can't add right now.

History & ratings

When on, rating a book removes it from your History & ratings list. When off, rated books sink to the bottom so you can still see and adjust them.

I've already read…

Tell Stackarr a book you've read so it shapes your suggestions — no download.

Import your reading list

Pull your "Want to Read" shelf in as suggestions.

How to find it: Goodreads → My Books → open your to-read shelf → scroll to the bottom → right-click the orange RSS icon → Copy link. Paste it here.

Hardcover → Settings → API → copy your token.
